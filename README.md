gochimp
=======

Go based API for Mailchimp, starting with Mandrill

to run tests, set a couple env variables:
```bash
$ export MANDRILL_KEY=111111111-1111-1111-1111-111111111
$ export MANDRILL_USER=user@domain.com
```
